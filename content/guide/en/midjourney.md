## Overview

Midjourney is one of the most popular AI image generation tools, renowned for exceptional artistic quality and aesthetics. It operates through Discord using text prompts and parameters.

## Core Features

| Feature | Description |
|---------|-------------|
| Text-to-Image | Generate images from natural language descriptions |
| Style Reference (--sref) | Reference an image URL to match its style |
| Character Reference (--cref) | Maintain consistent character appearance |
| Upscaling | Increase image resolution while preserving detail |
| Inpainting (Vary Region) | Selectively regenerate parts of an image |
| Pan and Zoom | Extend images beyond original boundaries |

## How to Use

1. Join via midjourney.com - redirects to Discord
2. Subscribe to a plan to start generating
3. Type /imagine followed by your prompt in a Discord channel
4. Use parameters: --ar 16:9 (aspect ratio), --s 750 (stylize), --c 50 (chaos)
5. Use U1-U4 to upscale, V1-V4 for variations

## Prompt Tips

- Be descriptive: subject, setting, lighting, style, mood, composition
- Reference art styles: "oil painting", "photorealistic", "anime", "3D render"
- Use style references (--sref URL) for specific look
- Experiment with stylize and chaos values
- Iterate systematically: start broad, narrow down with variations
