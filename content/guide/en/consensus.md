## Overview

Consensus is an AI-powered academic search engine that searches 200M+ scientific papers for evidence-based answers. It uniquely shows whether studies agree or disagree via the Consensus Meter.

## Core Features

| Feature | Description |
|---------|-------------|
| AI Research Search | Natural language queries against 200M+ papers |
| Consensus Meter | Visual indicator of study agreement/disagreement |
| Study Summaries | AI-generated one-sentence summaries of key findings |
| Quality Filters | Filter by RCT, systematic review, meta-analysis |
| Citation Export | Quick access to full citations |
| Paper Search | Find papers by title, author, or topic |

## How to Use

1. Visit consensus.app and sign up
2. Ask research questions in plain English
3. Review the Consensus Meter for scientific agreement
4. Filter by study design (RCT for highest evidence quality)
5. Export citations and save papers to your library

## Pros and Cons

**Pros**: Focused on peer-reviewed research, unique Consensus Meter, strong quality filters, affordable pricing.
**Cons**: Limited to academic papers, AI summaries can oversimplify nuanced findings, free tier limits AI queries.
