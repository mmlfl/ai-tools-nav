## Overview

ChatGPT is a conversational AI assistant developed by OpenAI, powered by large language models. Since its 2022 launch, it has become one of the most popular AI tools for learning, work, coding, and creative writing.

## Core Features

| Category | Capabilities |
|----------|-------------|
| Text Generation | Articles, emails, social copy, product descriptions in multiple styles |
| Coding Assistance | Python, JavaScript, and more - generation, debugging, optimization |
| Creative Writing | Novels, poetry, scripts, ad copy with structural suggestions |
| Learning Support | Explaining concepts, solving problems, creating study plans |
| File Upload (Plus) | PDF, Word, Excel analysis, extraction, and summarization |
| Custom GPTs | Create specialized GPTs for specific tasks and workflows |

## How to Use

1. Visit chat.openai.com and sign up with email or Google
2. Start a conversation by typing your question or request
3. Use advanced features: web search, DALL-E images, Code Interpreter
4. Upload files for analysis (Plus plan)
5. Organize conversations in the sidebar

## Pricing

| Plan | Price | Features |
|------|-------|----------|
| Free | $0 | GPT-4o mini, limited messages |
| Plus | $20/month | GPT-4o, DALL-E, browsing, file uploads |
| Pro | $200/month | Unlimited access, priority performance |
| Team | $25/user/month | Team workspace, admin controls |

## Pros and Cons

**Pros**: Wide capabilities from coding to creative writing, strong ecosystem (DALL-E, Code Interpreter, GPT Store), excellent multilingual support.
**Cons**: Free tier limitations, can occasionally produce inaccuracies, requires clear prompting for best results.
