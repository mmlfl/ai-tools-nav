## Overview

Claude is an advanced AI assistant developed by Anthropic, focused on safety, reliability, and deep understanding. It supports up to 200K tokens of context - upload entire novels, technical docs, legal contracts, or codebases.

## Core Features

| Feature | Description |
|---------|-------------|
| Long-form Processing | 200K token context for full document analysis |
| Code Understanding | Multiple languages - explanation, debugging, refactoring |
| Multi-turn Memory | Maintains context across complex multi-step tasks |
| Document Comparison | Side-by-side comparison and structured summarization |
| Safety Framework | Constitutional AI for reliable, unbiased outputs |

## How to Use

1. Visit claude.ai and sign in with Google or email
2. Type your question or upload files via the paperclip icon
3. Use advanced techniques: specify output formats, request step-by-step reasoning, role-playing
4. Manage conversations in the sidebar

## Pricing

| Plan | Price | Features |
|------|-------|----------|
| Free | $0 | Daily limits (~20-30 msgs), Claude Sonnet |
| Pro | $20/month | Unlimited messages, priority model access |
| Team | Custom | Multi-user, SSO, audit logs, API access |

## Pros and Cons

**Pros**: Industry-leading context window, high accuracy and safety, excellent file understanding, clear structured reasoning.
**Cons**: Free tier has significant daily limits, no official mobile app, Chinese support slightly less refined than English.
