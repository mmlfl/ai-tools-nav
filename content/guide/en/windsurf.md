## Overview

Windsurf is an AI-native IDE by Codeium, featuring the Cascade AI agent for complex multi-file operations. It provides a smooth, guided AI coding experience with less configuration needed.

## Core Features

| Feature | Description |
|---------|-------------|
| Cascade Agent | AI agent for multi-file editing, refactoring, and complex tasks |
| Tab Completion | Smart inline completions for code and edits |
| Inline Chat | Ask Cascade to modify selected code with diff preview |
| Codebase Awareness | Full project context maintained across sessions |
| Terminal Integration | AI-assisted command suggestions and explanations |

## How to Use

1. Download from codeium.com/windsurf
2. Start coding - Tab to accept suggestions
3. Open Cascade from sidebar for complex multi-file tasks
4. Describe what you want in natural language, review diffs, accept changes

## Pros and Cons

**Pros**: Smooth AI-native experience, Cascade handles complex tasks well, affordable Pro tier, less configuration needed.
**Cons**: Smaller community than Cursor, single model (Cascade only), newer product.
