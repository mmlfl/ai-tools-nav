## Overview

Leonardo AI combines ease of use with advanced features like custom model training. Built on Stable Diffusion, it adds a polished web interface, community models, and specialized tools for game asset creation.

## Core Features

| Feature | Description |
|---------|-------------|
| AI Image Generation | Text-to-image with multiple models and presets |
| Custom Model Training | Train models on your own images for consistent style |
| Community Models | Access thousands of user-trained models |
| Image Guidance | Use reference images to guide generation |
| Realtime Canvas | Draw and see AI generation in real-time |
| Alchemy | Enhanced photorealism and detail mode |

## How to Use

1. Sign up at leonardo.ai
2. Choose a model (Phoenix, Lightning, or community) and style preset
3. Enter prompt and optional negative prompt, generate
4. Train custom models: upload 10-30 images in consistent style
5. Use Realtime Canvas for interactive creation

## Pros and Cons

**Pros**: Easy-to-use web interface, built-in custom model training, large community model ecosystem, excellent for game assets, good balance of power and accessibility.
**Cons**: Credit limits can be restrictive, less customizable than local SD, internet required.
