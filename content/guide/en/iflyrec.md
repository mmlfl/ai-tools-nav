## How to Use

### Step 1: Account Setup and Basic Configuration
1. Visit [https://www.iflyrec.com](https://www.iflyrec.com) and register using your email or corporate SSO
2. Complete the onboarding wizard (select primary language, industry, and use case)
3. Install the Chrome extension for real-time web transcription (available in the Extensions menu)
4. Configure notification preferences in Settings > Notifications (enable email/SMS alerts for completed transcriptions)

### Step 2: Transcribing Audio/Video Files
1. **Upload Method**: 
   - Click "New Task" > "Upload File" on the dashboard
   - Drag and drop files (supports MP3, WAV, M4A, MP4 up to 5GB)
   - For video files: Check "Extract audio and transcribe" in processing options
2. **Processing Options**:
   - Select language (37 available) and dialect (e.g., Cantonese vs. Mandarin)
   - Enable "Speaker Separation" for multi-person recordings
   - Choose industry model (Medical, Legal, General Business)
3. **Post-Processing**:
   - Wait for processing (typically 1/4 real-time speed)
   - Click "Edit" to correct errors using AI suggestions (press Tab for next suggestion)
   - Use "Highlight" tool to mark key sections for sharing

### Step 3: Real-Time Transcription
1. **For Meetings**:
   - Open "Real-Time" tab and select meeting type (Internal/External)
   - Grant microphone access and test audio levels
   - Click "Start Recording" – text appears instantly with speaker labels
   - During meeting: Click "Pause" to skip irrelevant sections
2. **For Web Conferences**:
   - Enable the Chrome extension during Zoom/Teams calls
   - Select "Transcribe this tab" in the extension menu
   - View live transcript in side panel or full-screen mode
3. **Post-Meeting Actions**:
   - Click "Generate Notes" to auto-create structured summary
   - Assign action items by dragging text to "Tasks" section
   - Export as formatted PDF with meeting metadata

### Step 4: Advanced Workflow Integration
1. **Translation Workflow**:
   - In transcript view, click "Translate" > Select target language
   - Toggle between side-by-side and inline translation modes
   - Export bilingual document for professional review
2. **Collaboration Features**:
   - Share transcript via "Collaborate" button (set view/edit permissions)
   - Use @mentions to assign tasks directly in transcript
   - Enable version history to track all edits
3. **API Integration**:
   - Access API keys under Developer Settings
   - Use provided SDKs for Python, Java, and Node.js
   - Set up automatic transcription workflows via Zapier

**Pro Tips for Power Users**:
- Create custom vocabulary: Add industry terms in Settings > Custom Dictionary (improves accuracy by 25%)
- Use keyboard shortcuts: `Ctrl+Space` (pause), `Ctrl+M` (add marker), `Ctrl+T` (translate)
- Batch processing: Upload multiple files simultaneously and process with identical settings
- Mobile transcription: Use the iOS/Android app to record and transcribe on-the-go with cloud sync
