## Overview

Gamma is an AI-powered platform for creating beautiful presentations, documents, and web pages in minutes. It combines AI content generation with intelligent design as an alternative to PowerPoint and Google Slides.

## Core Features

| Feature | Description |
|---------|-------------|
| AI Generation | Create presentations, docs, web pages from prompts or outlines |
| Smart Design | Automatic layout, typography, and visual design |
| Templates | Professionally designed templates for various use cases |
| Real-time Collaboration | Multiple users can edit simultaneously |
| Analytics | Track who viewed your content and for how long |
| Brand Kit | Custom colors, fonts, logos (Business plan) |
| AI Images | Generate custom images within presentations |

## How to Use

1. Sign up at gamma.app
2. Click "Create New" - choose format (Presentation, Document, Web Page)
3. Create via: Paste text, Generate from prompt, or Import from files/URLs
4. Customize: edit text, change themes, add cards
5. Share as live web page, present, or export to PDF/PPTX

## Pricing

| Plan | Price/mo | Features |
|------|----------|----------|
| Free | $0 | 400 AI credits, basic features |
| Plus | $10 | Unlimited AI, remove Gamma branding |
| Business | $20/user | Brand kit, analytics, custom fonts, collaboration |

## Pros and Cons

**Pros**: Fast from idea to polished output, beautiful designs without skills, multiple formats, good collaboration, exports to PDF/PPTX.
**Cons**: AI content may need editing, limited animation vs PowerPoint, template-based design, internet required.
