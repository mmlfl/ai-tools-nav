## Overview

Perplexity is an AI-powered search engine combining LLM abilities with real-time web search. It provides direct answers with citations to source materials, making it ideal for research and fact-checking.

## Core Features

| Feature | Description |
|---------|-------------|
| AI Search | Natural language queries with direct, cited answers |
| Real-Time Web | Searches the live web for current information |
| Pro Search | Multi-step research with deeper analysis |
| Collections | Organize research into themed collections |
| Focus Modes | Filter to academic, video, social, or writing |
| File Upload | Analyze PDFs, images, and documents |
| Pages | Create shareable research pages |

## How to Use

1. Visit perplexity.ai - start searching immediately
2. Ask questions in natural language - be specific
3. Click source links to verify information
4. Use Focus modes for targeted searches (Academic, Video, Social)
5. Organize with Collections for different research topics

## Pricing

| Plan | Price/mo | Features |

## Pros and Cons

**Pros**: Fast accurate answers with citations, excellent for research, clean interface, strong Pro Search, good free tier.
**Cons**: Pro needed for heavy research, can surface low-quality sources, less suited for creative writing.
