## Overview

Runway is a comprehensive AI creative suite focused on video generation and editing. It combines generative AI (Gen-3) with professional video tools - motion tracking, green screen, inpainting, and more.

## Core Features

| Feature | Description |
|---------|-------------|
| Gen-3 Video Generation | Text-to-video and image-to-video |
| Motion Brush | Paint motion direction onto specific image areas |
| Green Screen | AI-powered background removal without a green screen |
| Inpainting | Remove or replace objects in video frames |
| Frame Interpolation | Generate frames between existing frames for slow motion |
| Text-to-Image | Still image generation (Gen-3) |

## How to Use

1. Sign up at runwayml.com
2. Start a project and select Gen-3 Alpha
3. Enter prompt or upload starting image, adjust settings
4. Use editing tools: Motion Brush, Green Screen, Inpainting
5. Export in various resolutions and formats

## Pros and Cons

**Pros**: Complete creative suite beyond generation, professional editing tools, Motion Brush innovation, good API, established platform.
**Cons**: 18-second video limit (Gen-3), credit system restrictive, quality varies for complex scenes, learning curve for full suite.
