## How to Use

### Step 1: Access and Setup
1. Visit [https://tongyi.aliyun.com](https://tongyi.aliyun.com) and click "Sign Up"
2. Register using your email or social accounts (Alibaba ID, WeChat, or international options)
3. Complete verification (email/SMS) – note: international users may need to use a VPN for SMS verification
4. Upon login, you'll see the main interface with a chat window, file upload button, and feature shortcuts

*Pro Tip:* For optimal experience, enable "Advanced Mode" in Settings to access all features. International users should set language to English under "Profile > Language Preferences."

### Step 2: Basic Interaction
1. Start with clear, specific prompts: Instead of "Explain math," try "Solve this calculus problem step-by-step: ∫(x²+3x)dx from 0 to 5"
2. Use the + button to upload images/PDFs: Qwen will automatically analyze visual content
3. For code tasks, specify language and requirements: "Write a Python function to sort this JSON data by date"
4. Add "Explain like I'm a beginner" to complex responses for simplified explanations

*Example Workflow:*  
Upload a financial report PDF → Type "Extract key metrics from pages 3-5 and create a comparison table" → Qwen processes the document, identifies relevant data, and formats it into a clean table.

### Step 3: Advanced Techniques
- **Math Problem Solving:** Prefix with "Use chain-of-thought reasoning for:" followed by the problem
- **Multimodal Tasks:** Combine image uploads with text: "This graph shows Q3 sales. Write a 3-paragraph analysis highlighting trends"
- **Code Debugging:** Paste error messages with "Debug this Python code and explain the fix"
- **Document Processing:** Upload multiple files and ask "Compare the project timelines in these documents"

*Pro Tip:* Use system prompts for consistent results. Start messages with:  
"[System: You are a senior data scientist. Provide detailed explanations with real-world examples.]"  
This customizes Qwen's response style for specific professional contexts.

### Step 4: Office Productivity Integration
1. **Meeting Notes:** Upload audio transcripts or type notes → Command: "Convert to formal minutes with action items"
2. **Report Generation:** Provide raw data → "Create a 10-slide presentation with charts and executive summary"
3. **Document Analysis:** Upload contracts → "Highlight key clauses and potential risks in a table"
4. **Email Drafting:** Type "Draft a professional email to [client] about [topic]" for instant templates

*Critical Workflow:* For business reports, combine multiple features:  
1. Upload sales data PDF  
2. Type: "Extract Q4 metrics, create a comparative analysis vs Q3, and generate 3 strategic recommendations"  
3. Use the "Export" button to save as Word/PDF
