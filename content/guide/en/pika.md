## How to Use Pika: Step-by-Step Guide

### Step 1: Account Setup & Interface Tour
1. **Sign Up**: Visit [pika.art](https://pika.art) and create an account (free tier available).  
2. **Dashboard Navigation**:  
   - *Create Tab*: Where all magic happens (default view)  
   - *History*: Access past generations (free users get 30-day history)  
   - *Community*: Browse trending videos for inspiration  
3. **Credit System**: Free accounts start with 100 credits (1 credit = 1 video generation). Earn 25 daily credits via login.  

### Step 2: Generating Your First Video (Text-to-Video)
1. **Enter Prompt**:  
   - *Basic*: "A glowing jellyfish floating in deep ocean"  
   - *Advanced*: "A glowing jellyfish (motion: slow upward float) in deep ocean (style: Cinematic), no fish, no text"  
2. **Adjust Settings**:  
   - *Style*: Select "Cinematic" from dropdown  
   - *Motion*: Use the slider (0-100); 40-60 for natural movement  
   - *Size*: 1024x576 (default) for landscape, 576x1024 for vertical Reels  
3. **Generate**: Click "Create" → Wait 45-90 seconds.  
4. **Refine**: If results are imperfect:  
   - Use "Vary" to regenerate similar versions  
   - Add negative prompts in the "Edit" tab (e.g., "no tentacle distortion")  

### Step 3: Image-to-Video Workflow
1. **Upload Image**: Click "Image" → "Upload" (supports PNG/JPG up to 10MB).  
2. **Add Motion Prompt**:  
   - Example for a landscape photo: "slow cloud movement, gentle river flow"  
   - *Critical*: Be specific about direction ("left-to-right wind")  
3. **Preserve Key Elements**:  
   - Use "Inpainting" to lock static areas (e.g., "keep the mountain unchanged")  
4. **Generate & Export**: Same as text-to-video; download as MP4 (max 1080p).  

### Step 4: Advanced Techniques for Professionals
- **Layered Motion Control**:  
  Prompt: "A robot (motion: walking forward) in a city (motion: camera zoom out), no pedestrians"  
  *Result*: Robot moves independently while background zooms.  
- **Frame-Level Editing**:  
  1. Generate video → Click "Edit" on a specific frame  
  2. Use the brush tool to mask areas → Describe changes (e.g., "change sky to sunset")  
- **Batch Variations**:  
  Add `[color: red, blue, green]` to prompts for automatic multi-color outputs.  
- **Pro Workflow**:  
  `Text prompt → Generate → Edit motion vectors → Add sound in CapCut → Export to TikTok`  

**Troubleshooting Common Issues**:  
- *Blurred Motion*: Reduce motion intensity (30-50) or simplify prompt  
- *Object Distortion*: Add negative prompts ("no melting", "no stretching")  
- *Slow Generation*: Use smaller canvas size (576x576 for testing)
