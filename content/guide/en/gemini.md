## Overview

Gemini is Google's multimodal AI assistant, deeply integrated with Google Search and Workspace (Gmail, Docs, Sheets). It supports text, image, audio, and video inputs.

## Core Features

| Feature | Description |
|---------|-------------|
| Multimodal Input | Text, images, audio, and video in a single conversation |
| Workspace Integration | Direct access to Gmail, Docs, Sheets, Drive |
| Web Search | Native Google Search for real-time answers |
| Long Context | Up to 1M tokens for very large documents |
| Image Generation | Built-in Imagen 3 |

## How to Use

1. Visit gemini.google.com and sign in with your Google account
2. Type questions, upload files, or use voice input
3. Use Workspace extensions for Gmail, Docs, Drive integration
4. Double-check responses with the "Google it" button

## Pricing

| Plan | Price | Features |

## Pros and Cons

**Pros**: Deep Google ecosystem integration, massive context window, native multimodal support, strong web search grounding.
**Cons**: Region-restricted features, coding slightly behind Claude/ChatGPT, privacy considerations.
