## How to Use

### Step 1: Account Setup and Interface Navigation
1. Visit [https://hailuoai.com](https://hailuoai.com) and sign up using your email or social account
2. Upon login, you'll see the dashboard with three main tabs: **Voice Studio**, **Music Lab**, and **Dialogue Center**
3. The left sidebar contains your project library, credit balance, and settings
4. Click "New Project" to start any workflow—each project type has its own guided setup

### Step 2: Creating Professional Voiceovers (Voice Studio)
1. In Voice Studio, enter your text in the editor (supports up to 5,000 characters per batch)
2. Select a voice from the 30+ options using the filter (e.g., "Female - Calm - English")
3. Customize parameters:
   - Speed: Adjust from 0.5x (slow) to 2.0x (fast)
   - Pitch: Slide to make voices higher/lower
   - Emotion: Choose from neutral, happy, sad, excited, or formal
4. For advanced control, switch to SSML mode and add tags like `<prosody rate="slow">` for emphasis
5. Click "Generate" and wait 10-30 seconds for processing
6. Use the built-in editor to trim silence, add crossfades, or adjust volume peaks
7. Download as WAV (lossless) or MP3 (compressed) when satisfied

*Pro Tip:* For long-form content, use the "Batch Processing" feature to upload multiple text files at once. The platform automatically preserves consistent voice parameters across all segments.

### Step 3: Generating Original Music (Music Lab)
1. Click "Create New Track" in Music Lab
2. Choose genre (e.g., "Cinematic", "Lofi Hip Hop", "Corporate")
3. Set parameters:
   - Duration: 15-180 seconds
   - Mood: Bright, melancholic, intense, etc.
   - Instrumentation: Select primary instruments
4. Toggle "Stem Separation" to get individual tracks for drums, bass, and melody
5. Click "Compose" and wait 20-60 seconds
6. Use the timeline editor to:
   - Trim sections
   - Adjust volume levels per stem
   - Add transitions between segments
7. Export as single track or separate stems for advanced mixing

*Pro Tip:* For video creators, use the "Sync to Video" feature by uploading your video file—the AI will automatically match music tempo to scene changes.

### Step 4: Building Intelligent Conversations (Dialogue Center)
1. Start a new dialogue session and select purpose:
   - Language Practice
   - Character Roleplay
   - Customer Service Simulation
2. Set conversation parameters:
   - Language pair (e.g., English to Spanish)
   - Personality traits (e.g., "formal", "friendly")
   - Knowledge domain (e.g., "medical", "technical")
3. Type your message or use voice input
4. The AI responds with both text and optional voice output
5. Use the "Memory" slider to control how much context the AI remembers
6. Export full transcripts with timestamps for analysis

*Pro Tip:* For language learning, enable "Slow Speech" in settings to get 30% slower voice output with clear pronunciation.

### Advanced Workflow: Creating a Multilingual Podcast Episode
1. Write your script in Voice Studio using SSML for emphasis
2. Generate English voiceover with "Podcast - Professional" voice
3. Use Dialogue Center to translate the script to Spanish with "Conversational" tone
4. Generate Spanish voiceover with matching emotional tone
5. In Music Lab, create background music with "Upbeat Corporate" style (120 BPM)
6. Use the integrated editor to:
   - Add intro/outro music
   - Balance voice/music volume
   - Insert pauses between segments
7. Export final mix with broadcast-ready audio levels
