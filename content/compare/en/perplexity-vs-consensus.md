## Overview

Perplexity is a general-purpose AI search engine for everyday queries. Consensus specializes in academic research, searching 200M+ scientific papers for evidence-based answers.

## Feature Comparison

| Feature | Perplexity | Consensus |
|---------|------------|-----------|
| Search Scope | Web-wide | Academic papers (200M+) |
| Source Type | News, blogs, websites | Peer-reviewed research |
| Consensus Meter | No | Yes (study agreement) |
| Study Quality Filters | No | Yes (RCT, systematic review) |
| Pro Pricing | $20/month | $8.99/month |

## Verdict

Perplexity for everyday research and current events. Consensus for academic, medical, and evidence-based research where source quality matters. Many researchers use both.
